# CodeAlpha Internship - Personal Portfolio Web Page

A sleek, modern, and fully responsive **Personal Portfolio** website built during my Web Development Internship at **CodeAlpha**. This project features a dual-column layout with a persistent profile sidebar and a dynamic main content area to showcase my technical skills, education, and development journey.

## 🚀 Live Demo

Check out the live deployment of the project here:  
👉 **[Live Demo Link](https://mahammad-rahil-vahora.github.io/codealpha_tasks/Portfolio/)**

---

## 📌 Features

* **Profile Sidebar:** A dedicated professional card featuring a profile picture, quick contact details (Email, Phone, Location), and social connectivity icons (GitHub, LinkedIn).
* **About Me:** A clean introduction highlighting my background as a student passionate about front-end web development, problem-solving, and building practical applications.
* **Skills Inventory:** Visual grid showcasing proficiency across key technologies, including:
  * Front-End: **Html5, Css3, JavaScript**
  * Languages & Databases: **Python, MySQL**
  * Tools: **Git & Github**
* **Education History:** A structured breakdown of my academic path and qualifications.
* **Projects / Portfolio:** A curated section displaying tasks completed during my front-end journey.
* **Modern Dark Theme:** A premium, eye-catching dark mode user interface featuring smooth contrast highlights and clean container borders.

---

## 🛠️ Tech Stack

* **HTML5:** Structured semantic markup for content sections, skill containers, and contact info cards.
* **CSS3:** Styled the page layout, implemented custom font families, rounded containers, uniform grid spacing, and responsive alignment.
