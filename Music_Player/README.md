# CodeAlpha Internship - Music Player Web Application

A feature-rich, sleek, and interactive **Music Player** web application built during my Web Development Internship at **CodeAlpha**. This project highlights front-end logic, working with media APIs, and creating dynamic user interfaces with real-time controls.

## 🚀 Live Demo

Check out the live deployment of the project here:  
👉 **[Live Demo Link](https://mahammad-rahil-vahora.github.io/codealpha_tasks/Music_Player/)**

---

## 📌 Features

* **Audio Controls:** Core playback features including Play, Pause, Next, and Previous track controls.
* **Progress Bar:** Interactive progress slider that displays the current playback time and allows users to seek to different parts of the song.
* **Volume Control:** Built-in volume slider to easily adjust or mute audio levels.
* **Track Details:** Displays dynamic textual metadata including the song title and artist name.
* **Playlist/Queue Management:** Supports transitioning smoothly between multiple tracks in the playlist.
* **Responsive & Modern UI:** A beautiful user interface styled with clean layouts and smooth transitions, fully optimized for both desktop and mobile viewports.

---

## 🛠️ Tech Stack

* **HTML5:** Built the layout for the player interface, tracking controls, and audio wrappers.
* **CSS3:** Styled the application with interactive hover animations, custom slider track stylings, and mobile-responsive layouts.
* **JavaScript (ES6):** Utilized the HTML5 Audio API to control playback logic, manage track listings, and update temporal UI elements in real-time.
